# Roles & Responsibilities

SmartPAL is restricted to only making changes to Canvas elements that are directly related to SmartPAL. 

| Canvas Action | SmartPAL Team | Instructor/TA |
|---------------|:-------------:|:-------------:|
| Create/manage SmartPAL Module | ✓ | |
| Upload/update SmartPAL documents (Student Guide, IRB Information Sheet, etc.) | ✓ | |
| Create SmartPAL assignment groups | ✓ | |
| Create/update SmartPAL assignments with appropriate point values | ✓ | |
| Set/change due dates for SmartPAL assignments | ✓ | |
| Set/change SmartPAL gradebook presence | ✓ | |
| Publish/unpublish/hide/release SmartPAL related grades | ✓ | |
| Post SmartPAL-related grades | ✓ | |
| Update students about SmartPAL research participation via Canvas Inbox and/or SmartPAL apps | ✓ | |
| Email students about SmartPAL research participation via SmartPAL User Account (rarely) | ✓ | |
| Answer SmartPAL related student inquiries | ✓ | |
| Answer SmartPAL related instructor inquiries | ✓ | |
| Manage non-SmartPAL Canvas elements (content, grades, assignments etc) | ✗ | ✓ |
| Post Canvas announcements | ✗ | ✓ |
| Post on Canvas message boards | ✗ | ✓ |
